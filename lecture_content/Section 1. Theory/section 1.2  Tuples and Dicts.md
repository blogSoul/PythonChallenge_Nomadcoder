## Section 1.2 Tuples and Dicts

list는 common하고 mutable한 객체에만 적용이 가능하고
Tuple은 immutable한 객체에만 적용이 가능합니다.
Ex)```days = ("Mon", "Tue", "Wed", "Thur", "Fri")```

dictionary은 key와 그에 해당되는 객체를 가지고 있습니다.
Ex)
```
Soul_M = {
    "name" : "Soul",
    "age" : 21,
    "korean" : True,
    "fav_food" : ["Kimchi", "Sam"]
}
print(Soul_M)
```
